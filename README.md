# 💰 AI Powered Finance Manager

An intelligent web-based finance management application built using **HTML, CSS, and JavaScript**.  
This app helps users track expenses, manage income, and gain financial insights with smart analysis features.

---

## 🚀 Features

- 📊 Add and track income & expenses
- 🧾 Categorize transactions
- 📅 View transaction history
- 📈 Real-time balance calculation
- 🧠 AI-based financial insights (basic rule-based logic)
- 💾 Data stored using Local Storage
- 📱 Responsive design (Mobile + Desktop)

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling & Layout
- **JavaScript (Vanilla JS)** – Functionality & Logic
- **Local Storage API** – Data persistence


---

## 📸 How It Works

1. User adds income or expense
2. App updates total balance automatically
3. Transactions are stored in local storage
4. AI logic analyzes spending patterns
5. Smart suggestions are displayed

---

## 🧠 AI Logic (Basic Version)

The AI system:
- Detects high spending categories
- Warns if expenses exceed income
- Suggests saving tips based on patterns

Example:
- If Food expenses > 40% of income → Suggest budget adjustment
- If savings < 20% → Recommend saving strategy
